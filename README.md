# keycloak-demo

## my-java-app Project setup

```
./gradlew bootRun
```

## my-vue-app Project setup

Start up Vue application using the Vue CLI development server
```
yarn install
```

## Project startup
```
yarn serve --port 8081
```


### Start Keycloak
```
./kc.sh start-dev
```

### Start Keycloak Using Docker
#### Build and run the Docker container
```
docker-compose up
```
